# Flux - Notion Style Kanban Board

A minimal, strict dark-mode Notion-like Kanban board built with React, Vite, and Node.js.

## Features
- **Hover Menus:** Nested right-click and action menus (ex: Open In, Color Picker).
- **Inline Editing:** Double click to edit any card title or column header instead of separate modals.
- **Strict Dark Theme:** #191919 background. No frosted glass blur effects or neon glows. Flat design language.
- **Collapsible Side Panel:** Expands for specific card property editing.

## To Run:
1. `npm install`
2. `npm run dev`
